Graduaat Proef

De opdracht werd in groep gemaakt door samenwerking Ali Maes, Ahmet Yimaz en Filip Rigoir.
De applicatie is geprogrammeert in C# met verschillende onhafhankelijke lagen. Er wordt voor elk nieuw project binnen de solution gebruik gemaakt van .NET Core 5.0. We gebruiken als debuger Microsoft Visual Studio. De opstart van het project kent een minimale configuratie die we kort bespreken hoe het project tot stand is gekomen. Onderstaande acties werden systematisch uitgevoerd wanneer de app daar klaar voor was om de volgende stap in te gaan.
