Stage Project

Ik heb als stagiair op hetproject AvanadeAsset Management gewerkt. Dit project ging over het managen van assets overal in de wereld.
Waarbijde klant de status van een asset of warehouse kan managen, toevoegen, updaten of verwijderen.

API

De API-layerwas gemaakt met .Net 6 framework.De back-end zorgde voor de CRUDL operatie voor de bewerkingen van data
(Create, Read, Update, delete, List).

React

Nadat het grootste gedeelte van API gemaakt was, zijn we overgestapt naar front-end.
We moesten met Microsoft fluent ui werken voor het design.
Ik moest ook een map voorzien voor de home page, waar we de warehouses en assets konden zien en op zoeken.

Azure

Naast het bouwen van API en front-end hebben we ook heel wat dingen gedaan met Azure. Bv:
Het opladen van het API op een web app service, het opladen van front-end-web op een static web app service,
azure pipelines leren schrijven op een yaml file (Voor testing, building, running, deployment of webapps enz...).

IOT

De volgende fase was het werken met IOT device, dit was heel nieuw voor mij, Ik moest met RaspberryPI werken,
de bedoeling was om telemetrie berichten van de sensors uit te lezen en dit aan te tonen op de web applicatie.
