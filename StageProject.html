Avanade Internship Project

During my internship at Avanade Belgium, I had the opportunity to work on a comprehensive enterprise asset management solution.

Key Achievements:

  • Developed a scalable microservices architecture using .NET 6
  • Implemented real-time asset tracking with SignalR
  • Created responsive front-end using React and TypeScript
  • Integrated Azure cloud services for deployment and monitoring
  • Implemented automated testing and CI/CD pipelines

Technology Stack:

  • .NET 6 Web API
  • React + TypeScript
  • Azure Cloud Services
  • SQL Server
  • Entity Framework Core
  • Azure DevOps
  • Docker

The project focused on creating a modern, scalable solution for managing and tracking enterprise assets across multiple locations.
