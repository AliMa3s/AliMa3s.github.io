Bibliotheek Management Systeem

Dit is een schoolproject waarin ik een bibliotheekapplicatie moest maken.

In deze applicatie kan je een boek lenen en teruggeven, de data word opgeslaan in Sql databank.

Daarnaast heb je ook toegang tot alle leden met hun informatie.

De informatiewijziging gebeurt in realtime met SQL-Server en wordt in realtime getoond in de applicatie.

Men kan leden selecteren en de gewenste boeken uitlenen of teruggebrachte boeken ingeven.

Een totaal overzicht van alle leningen worden ook getoond door de sql-server te verbinden met de databank.

Technologies:

  • Java
  • SQL Server
  • JDBC
  • Java Swing
