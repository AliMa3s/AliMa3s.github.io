Bibliotheek lenen van boeken en terugbrengen Gemaakt in Java

Dit is een school project waar ik moest een bibliotheek applicatie maken.

In deze applicatie kan je een boek lenen en uitgeven en de data word opgeslaan in Sql databank.
Daarnast heb je ook toegang tot alle leden met hun informatie.

De informatie wijziging gebeurt in realtime met SQL-Server. en word getoond in realtime in de applicatie.

Men kan een leden selecteren en de gewenste boek uitlenen en of teruggebrachte boek in geven.

Een totaal overzicht van alle leningen worden ook getoond via sql-server verbinden met databank.



Voor meer info bezoek mijn github aub.
