Ava Asset Management (c# ASP.NET, React-typscript, Azure )

Ik heb als stagiair op het project Avanade Asset Management gewerkt. Dit project ging over het managen van assets overal in de wereld.
Waarbijde klant de status van een asset of warehouse kan managen, toevoegen, updaten of verwijderen.
Deze applicatie moet het beheer toelaten van klanten, truitjes en bestellingen.
Gebruikt technologie :
•ASP.NET Core 6.0
•Azure Resources
•SQL Server
•xUnit
•EntityFramework
•React met typescript
Er moest gebruik gemaakt worden van een gelaagde architectuur met daarbij een datalaag, business-laag en user interface-laag.
