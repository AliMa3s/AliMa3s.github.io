Verkoop VoetbalTruitje (c# ADO.NET)

Er werd gevraagd om een desktop applicatie te ontwikkelen die ons moest helpen om voetbaltruitjes te verkopen.
Deze applicatie moet het beheer toelaten van klanten, truitjes en bestellingen.
RequirementsTe gebruiken technologie :
•.NET Core
•Visual Studio
•SQL Server
•xUnit
•ADO.NET
•WPF
Er moest gebruik gemaakt worden van een gelaagde architectuur met daarbij een datalaag, business-laag en user interface-laag.
