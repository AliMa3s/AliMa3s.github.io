AdresBeheer Applicatie Gemaakt in C#

Dit is een school project waar ik moest een AdresBeheer applicatie bouwen.

In deze applicatie wordt de data gelezen van een gml file met logica als backend
en achteraf al die data wordt verstuurd naar de databank.

In deze applicatie kan men adressen,gemeente,straat toevoegen, verwijderen en updaten.
Daarnast er zijn ook unittesten geschereven

Daarnast voor het gemaak van user is er een user interface gemaakt waarin kan een user met
bijna geen informatica achtergrond een adres kan gemakkelijk wijzigen.

Een totaal overzicht van alle adressen worden ook getoond via sql-server verbinden met databank in realtime.

Voor meer info bezoek mijn github aub.

Technologies:
  • - C# ADO.NET
