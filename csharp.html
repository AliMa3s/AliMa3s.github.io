AdresBeheer Application

Dit is een school project waar ik een AdresBeheer applicatie moest bouwen.

In deze applicatie wordt de data gelezen van een gml file met logica als backend en achteraf al die data wordt verstuurd naar de databank.

In deze applicatie kan men adressen, gemeente, straat toevoegen, verwijderen en updaten. Daarnaast er zijn ook unittesten geschereven.

Er is een gebruiksvriendelijke interface gemaakt waarbij gebruikers zonder technische achtergrond eenvoudig adressen kunnen beheren.

Een totaal overzicht van alle adressen worden in realtime getoond via de SQL Server database connectie.

Technologies:

  • C# ADO.NET
  • SQL Server
  • Windows Forms
  • Unit Testing
  • GML File Processing
