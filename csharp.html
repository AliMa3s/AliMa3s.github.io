AdresBeheer Applicatie Gemaakt in C#

Dit is een school project waar ik een AdresBeheer applicatie moest bouwen.

In deze applicatie wordt de data gelezen van een gml file met logica als backend
en achteraf al die data wordt verstuurd naar de databank.

In deze applicatie kan men adressen, gemeente, ,straat toevoegen, verwijderen en updaten.
Daarnaast er zijn ook unittesten geschereven

Daarnaast is er voor het gemak van de gebruiker een userinterface gemaakt waarin een gebruiker met
weinig informatica-achtergrond een adres gemakkelijk kan wijzigen.

Een totaal overzicht van alle adressen worden ook getoond door de sql-server te verbinden met de databank in realtime.

Voor meer info bezoek mijn github aub.

Technologies:
  • - C# ADO.NET
